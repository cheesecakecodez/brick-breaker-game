# Brick Breaker Game (Java)

A Brick Breaker game developed in **Java using Eclipse**, created as a learning-focused project to understand and implement **Object-Oriented Programming (OOP)** concepts through practical application.

The emphasis of this project is on **discipline, structure, and learning through implementation**.

---

## Project Intent

This project was built to:
- Strengthen core Java fundamentals
- Apply OOP concepts in a real, working program
- Understand interaction between multiple classes
- Improve logical thinking and debugging skills

---

## Object-Oriented Concepts Applied

- **Encapsulation**  
  Game data and behavior are confined within clearly defined classes.

- **Inheritance**  
  Shared properties and logic are reused through parentâ€“child class relationships.

- **Polymorphism**  
  Method overriding enables different behaviors for related objects.

- **Abstraction (introductory level)**  
  Focus on *what* an object does rather than *how* it does it.

## Features

- Keyboard-controlled paddle movement
- Ball movement and collision detection
- Brick destruction logic
- Score tracking
- Game over and restart handling

---

## Technology Used

- Language: Java
